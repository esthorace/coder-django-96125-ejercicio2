- Crear una app "servicios"

- Crear el modelo Cliente

    * nombre
    * apellido
    * celular
    * domicilio

- Crear el modelo Servicio

    * nombre
    * descripcion
    * precio
    * disponible

- Crear el modelo Pedido

    * (investigar TextChoices) para 3 estados: PENDIENTE, EN_PROGRESO, COMPLETADO

    * cliente_id
    * servicio_id
    * descripcion
    * fecha_solicitud
    * fecha_finalización
    * estado (PENDIENTE...)

- Registrar en admin y hacer pruebas

- Crear template base.html

- Crear vista para mostrar Clientes, Servicios y Pedidos
